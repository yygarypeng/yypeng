Projects

Physics Motivation

Quantum Correlations

Problem: Quantum-correlation observables in $H \to WW^* \to \ell\nu\ell\nu$ connect Higgs measurements to the structure of electroweak interactions, but the two neutrinos hide key rest-frame information.

Contribution: I study how W-boson polarization and density-matrix-sensitive observables can be accessed with realistic reconstruction and validation constraints.

Case Study

ML Event Reconstruction

Problem: Standard kinematic reconstruction is under-constrained in a final state with two unobserved neutrinos, yet the analysis needs rest-frame variables for interpretation.

Contribution: I develop transformer, flow-based, and physics-constrained models with reproducible training, ONNX export, systematic stress tests, and validation outputs that can be compared across ATLAS channels.

Collaboration Software

HGTD Production Database

Problem: Detector production requires reliable data management, quality-control views, and interfaces that multiple teams can use consistently.

Contribution: I co-develop frontend and backend components for the HGTD production-database webpage, helping collaborators organize, query, and visualize detector information during production.

Run 3 Analysis

Run 3 HWW Coupling

Problem: Run 3 measurements need analysis continuity, portable environments, and mentorship structures that let new contributors become productive quickly.

Contribution: I co-mentor two MSc students studying electroweak couplings in $H \to WW^*$ and contribute workflow tooling that reduces setup friction for collaborators.
